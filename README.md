# ♻️ Amiguinhos da Natureza: Uma Aventura de Reciclagem

> **Projeto de conscientização ambiental universitaria, alinhado à ODS 11 da ONU.**

Este é um projeto interativo desenvolvido com foco na **educação ambiental lúdica**. Através de uma interface acolhedora, os jogadores auxiliam personagens carismáticos a realizar a triagem correta de resíduos sólidos urbanos de uso cotidiano (Papel, Plástico, Metal e Vidro), acumulando pontos e sequências de acertos.

## 🎨 Características do Jogo

- **Estética Acolhedora (*Cozy Art*):** Tema lúdico com personagens carismáticos que promovem um ambiente de aprendizado calmo e sem pressões.
- **Sorteio Dinâmico de Personagens:** Sorteio de personagens no início da partida a partir de um banco de dados unificado.
- **Painel HUD:** Exibição de pontos e sequência de acertos.
- **Sistema de Áudio:** Emissão de áudio para acertos e erros.

---

## 🎮 Como Jogar

0. Preview page: https://station137.github.io/amiguinhos-da-natureza/

1. Um objeto reciclável aparecerá na mão do ursinho.
2. **Dica:** Se tiver dúvidas de qual objeto se trata, clique sobre ele para revelar a etiqueta com o nome.
3. Se quiser receber uma dica do seu Amiguinho da Natureza, de como jogar, basta clicar sobre o ursinho para abrir seu balão de diálogo.
4. Escolha e clique em um dos 4 cartões de lixeira na parte inferior (**Papel**, **Plástico**, **Metal** ou **Vidro**).
5. O jogo registrará sua resposta, emitirá o feedback visual e sonoro, e passará para o próximo objeto.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web puras (*vanilla*) para garantir excelente desempenho e facilidade de manutenção:

- **HTML5:** Estruturação semântica das camadas de jogo e acessibilidade.
- **CSS3:** Estilização responsiva, layout em CSS Grid, controle de proporções via `aspect-ratio` e animações de transformações chaveadas (`@keyframes`).
- **Vanilla JavaScript:** Gerenciamento do estado do jogo (pontuação, sequência, sorteios de objetos), efeito de paralaxe vetorial e acionamento dinâmico da API de Áudio HTML5.
